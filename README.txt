Данный репозиторий пренадлежит пользователю linerStalker. Открытый репозиторий включает в себя шейдеры из Gunslinger mod, которые оптимизируются мною по сей день.

ИНСТРУКЦИЯ ПО УСТАНОВКЕ:
	1. Скачайте Gunslinger mod, версию от 23.02.2022 (если есть версия актуальнее, то нужно проверить на совместимость.)
	2. Очистите папку shaders в папке gamedata.
	3. Клонируйте (НЕ СКАЧИВАТЬ АРХИВОМ!!!) репозиторий, поместить в папку shaders.
	4. По пути настроек игры пользователя, удалите папку shaders_cache, если таковая имеется.
	
ДОПОЛНИТЕЛЬНО:
	Если Вы найдете какие-либо баги, которые закономерно появились после установки оптимизированных шейдеров, то обязательно сообщите мне (linerStalker) и создайте проблему во вкладке репозитория Issues.

	4. In the user's game settings path, delete the 'shaders_cache' folder, if any.
